## Installation and Setup Instructions

Clone down this repository. You will need `node`, `npm` and `yarn` installed globally on your machine.  

Installation:

`yarn`  

To Run Test Suite:  

`yarn test`  

To Start Server:

`yarn start`  

To Visit App:

`localhost:3000/`
